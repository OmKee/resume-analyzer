# Resume Analyzer

A simple Python project that analyzes a resume (PDF) and compares it with a job description to calculate an ATS-style match score based on required skills.

## Features
- Extracts text from PDF resume
- Reads job description from a text file
- Identifies required skills
- Compares resume with job requirements
- Calculates ATS match score

## Tech Stack
- Python
- PyPDF2

## Project Structure
- main.py → main program logic
- resume.pdf → sample resume file
- job_description.txt → sample job description
- requirements.txt → dependencies
